Basic Operations

Addition (+):

The addition function is used to perform basic addition operations between two or more numbers.

Subtraction (-):

The subtraction function is used to perform basic subtraction operations between two numbers.

Division (/):

The division function is used to perform basic division operations between two numbers.

Advanced Operations

Exponentiation (^):

The exponentiation function is used to raise a number to a specific power. For example, 2^3 would be calculated as 2 raised to the power of 3 (2 * 2 * 2 = 8).

Square root (√):

The square root function is used to calculate the square root of a number. For example, the square root of 9 is 3.

Logarithm (log):

The logarithm function is used to calculate the logarithm of a number to a specific base. The most commonly used base is 10. For example, log10(100) = 2.

Natural logarithm (ln):

The natural logarithm function is used to calculate the logarithm of a number to the base e (Euler's number, approximately 2.71828). For example, ln(10) ≈ 2.30259.

Factorial (!):

The factorial function is used to calculate the factorial of a non-negative integer. The factorial of a number is the product of all positive integers less than or equal to that number. For example, 5! = 5 * 4 * 3 * 2 * 1 = 120.

Trigonometric Functions

Sine (sin):

The sine function is used to calculate the sine of an angle given in radians.

Cosine (cos):

The cosine function is used to calculate the cosine of an angle given in radians.

Tangent (tan):

The tangent function is used to calculate the tangent of an angle given in radians.

Inverse sine (asin):

The inverse sine function is used to calculate the angle whose sine is a given number.

Inverse cosine (acos):

The inverse cosine function is used to calculate the angle whose cosine is a given number.

Inverse tangent (atan):

The inverse tangent function is used to calculate the angle whose tangent is a given number.

Hyperbolic Functions

Sineh (sinh):

The hyperbolic sine function is used to calculate the hyperbolic sine of a number.

Cosineh (cosh):

The hyperbolic cosine function is used to calculate the hyperbolic cosine of a number.

Tangenth (tanh):

The hyperbolic tangent function is used to calculate the hyperbolic tangent of a number.

Inverse sineh (asinh):

The inverse hyperbolic sine function is used to calculate the inverse hyperbolic sine of a number.

Inverse cosineh (acosh):

The inverse hyperbolic cosine function is used to calculate the inverse hyperbolic cosine of a number.

Inverse tangenth (atanh):

The inverse hyperbolic tangent function is used to calculate the inverse hyperbolic tangent of a number.

Other Functions

Absolute value (abs):

The absolute value function is used to calculate the absolute value (magnitude) of a number, disregarding its sign.

Rounding (round):

The rounding function is used to round a number to the nearest whole number or a specified decimal place.

Modulo (mod):

The modulo function is used to calculate the remainder of the division between two numbers. For example, 10 mod 3 = 1.

Percentage (%):

The percentage function is used to calculate a percentage of a given number.

Random number (rand):

The random number function is used to generate a random number within a specified range.

Pi (π):

The Pi function returns the value of the mathematical constant π (pi), which represents the ratio of a circle's circumference to its diameter (approximately 3.14159).
